Frequently asked questions

1

Can I connect Quidity to my existing sales tools?

Yes. Quidity offers 28+ native integrations spanning CRM, email, social, dialer, AI, and automation categories. For tools without a direct integration, you can use Zapier (8,000+ apps), the REST API, or webhooks to build custom connections. Most native integrations take less than five minutes to set up.

2

How many integrations does Quidity have?

Quidity has 28+ direct integrations across nine categories: CRM, Email, Social & Messaging, AI & Developer, Automation, Parallel Dialer, Dialer, Video & Gifting, and Security. Through Zapier, you can connect Quidity to over 8,000 additional tools.

3

What CRM does Quidity integrate with?

Quidity integrates natively with Salesforce and HubSpot through two-way API connections that sync contacts, leads, activities, and pipeline data in real time. You can also connect Pipedrive and other CRMs through Zapier. Both the Salesforce and HubSpot integrations include automatic activity logging, field mapping, and exclusion list management so your CRM always reflects the latest outreach activity.

4

Can I use Quidity with ChatGPT or Claude?

Quidity is one of the first sales platforms to support the Model Context Protocol (MCP), which lets you search prospects, enrich contacts, pull activity history, and manage lead lists directly inside ChatGPT and Claude. Setup takes under a minute. Just sign in with your Quidity account when prompted. No API keys required.

5

Does Quidity have an API?

Quidity provides a full REST API with endpoints for contacts, sequences, lead lists, calls, tasks, mailboxes, and users. The API uses bearer token authentication and supports outbound webhooks for real-time activity notifications. Full documentation is available at docs.quidity.com.

6

Is the Quidity API free to use?

The REST API is included with your Quidity subscription at no additional cost. API access uses bearer token authentication and there are no per-call fees. Webhook notifications for activity events are also included. Check docs.quidity.com for rate limits and endpoint details.
